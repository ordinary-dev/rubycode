## How to run solutions

For programs written in C:

```bash
cd problems/random-problem
make
make run
```

For programs written in Rust:

```bash
cd problems/random-problem
cargo test
```

## Solution structure

For programs written in C:

- `lib.c` - the solution itself
- `main.c` - tests

For programs written in Rust:

- `src/lib.rs` - solution and tests
