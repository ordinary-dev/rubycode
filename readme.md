## How to run solutions

For programs written in Rust:

```bash
cd problems/random-problem
cargo test
```

For programs written in Ruby:

```bash
cd problems/random-problem
ruby test.rb
```

## Solution structure

For programs written in Rust:

- `src/lib.rs` - solution and tests

For programs written in Ruby:

- `main.rb` - the solution itself
- `test.rb` - tests
